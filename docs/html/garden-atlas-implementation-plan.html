Verification

  1. npm run typecheck validates shared, web, and mobile TypeScript.
  2. npm run test:web validates web behavior, including recommendation deck advance.
  3. npm run build:web validates the Vite production build.
  4. Browser QA confirms the first viewport shows one dominant plant card and all screens are reachable.
