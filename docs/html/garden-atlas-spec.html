Product Positioning

Garden Atlas is an aesthetic AI botanical memory archive. It is not a diagnosis tool, agricultural system, or strict taxonomy product. The emotional goal is to turn everyday encounters with plants into collectible botanical artworks.

MVP Scope

In Scope

  • Onboarding, home, capture, generation, result, gallery, detail, recommendation, and location screens.
  • Mock AI specimen generation with progress states.
  • Style modes, notes, favorites, tags, filters, opt-in recommendation sharing, and comparison slider.
  • Expo scaffold with matching route names and shared model shapes.

Deferred

  • Real OpenAI calls, auth, payments, cloud sync, production map SDK keys.
  • Real plant classifier, chat/follow graph, social feed, AR viewing, and plant care assistant.
  • Print-on-demand fulfillment and collaborative collections.

Core Screens

Home

Premium calm entry point with one dominant 封面植物 card on the first viewport. Swipe left to pass, right to like, then the next public plant loads into the same position.

API Contracts

  • POST /api/plant-entries creates a memory entry around an uploaded photo.
  • POST /api/plant-entries/{entryId}/generations starts specimen card generation.
  • GET /api/generations/{generationId} reads generation progress and output.
  • PATCH /api/plant-entries/{entryId} edits names, notes, tags, and favorite state.
  • GET /api/plant-entries powers the gallery with filters and search.
  • GET /api/recommendations/cover-plants returns opt-in public plant cards for the deck.
  • POST /api/recommendations/{entryId}/interaction records like or pass.
  • PATCH /api/me/recommendation-settings updates the user's opt-in sharing preference.

Production OpenAI calls stay server-side. The backend adapter can use the Responses API with the image generation tool for multi-step/iterative flows, or the Images API for single image generation and editing. References: OpenAI image generation guide, image generation tool guide, and Images API reference.

Recommendation Privacy

Default Closed

Entries stay private unless the user enables 开放我的图鉴推荐给他人 and marks cards as recommendable.

Public-Safe Data

Recommendations show generated artwork, plant names, broad location, style mode, tags, and display name. Precise GPS and private notes stay hidden.
