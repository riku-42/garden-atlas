手机预览方案

Expo 的 --tunnel 当前会在 ngrok 认证或 hostname 权限阶段失败。项目现在提供一个替代命令：先打开 Cloudflare quick tunnel，再把公开地址交给 Expo 生成可扫码的 QR。

推荐命令

cd "C:\Users\zhentaolu\OneDrive - Microsoft\Documents\New project\garden-atlas"
npm --workspace @garden-atlas/mobile run start:phone -- --clear

命令启动后，终端会先显示 Expo Go URL，并由 Garden Atlas 启动脚本直接打印一个 QR code。 用手机上的 Expo Go 扫码即可；手机不需要和工作电脑在同一个局域网。

保留的本地模式

npm --workspace @garden-atlas/mobile run start -- --lan --clear
npm --workspace @garden-atlas/mobile run start -- --localhost --clear

--lan 适合同一 Wi-Fi；--localhost 适合模拟器或本机浏览。跨网络手机预览优先使用 start:phone。

验证

新脚本有 smoke test：它会启动 Cloudflare tunnel 和 Expo，确认本地 /status 与公网 tunnel /status 都可访问，然后自动退出。 

npm --workspace @garden-atlas/mobile run start:phone -- --smoke-test --clear
